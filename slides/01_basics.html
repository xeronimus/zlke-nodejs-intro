01 Basics

Node.js

"...is an open-source, cross-platform, back-end, JavaScript runtime environment that executes JavaScript code outside a web browser."

  • https://nodejs.org/
  • https://en.wikipedia.org/wiki/Node.js

Node.js REPL

          
            $ node
            > let v = 0
            > ++v
            1
            > ('b'+'a'+ +'a'+'a').toLowerCase()

Node.js runtime

          
            // simple.js
            const greeting = 'Hello World!';
            console.log(greeting);
          
          
          
            $ node simple.js

Node.js core modules

No browser sandbox, we have access to file system, networking through core modules

Node.js history

Node.js was written initially by Ryan Dahl in 2009

npm was introduced in 2010
