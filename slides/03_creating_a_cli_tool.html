03 Creating a CLI tool

The "process" object

process.arg gives us access to the command line arguments


          // mySuperFile.js
          console.log(process.argv);
          

          $ node mySuperFile.js arg1 arg2
          

          [
            '/node/v10.16.2/bin/node',
            '/home/user/path/main.js',
            'arg1',
            'arg2'
          ]

The "process" object

process.exit allows us to abort. We can pass in the exitCode.


          // justSomeFile.js
          process.exit(1);
